# death

kill command wrapper. say "DEATH" with death voice.

## INSTALL

```
$ gem install death
```

## SYNOPSIS

```
death [-s signal_name] pid ...
death -l [exit_status]
death -signal_name pid ...
death -signal_number pid ...
```

### shortcut command

```
lml
```

## Operating environment

Mac OS X (only)
